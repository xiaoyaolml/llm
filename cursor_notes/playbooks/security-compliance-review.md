# Cursor Playbook: 安全 / 合规审查

## 适用场景

- 认证、授权、会话、审计改动
- 配置安全与敏感信息处理
- 金融或审计要求较强的代码路径

## 推荐提示词

```text
+先不要修改代码。
+请从安全和合规角度审查这部分改动：
+1. 输入验证与注入风险
+2. 认证与授权影响
+3. 敏感信息处理
+4. 审计日志与追踪影响
+5. 配置与默认值风险
+6. 合规性影响
+7. 修复优先级
+```
+
## 必查项
+
+- 是否存在注入、越权、敏感信息泄露
+- 是否破坏审计链路
+- 是否改变了访问控制边界
+- 是否写入了不安全默认配置
+- 是否使用不安全的字符串拼接、SQL、shell 调用
+
## 输出建议
+
+要求 Cursor 区分：
+
+- 立即修复项
+- 上线前必须修复项
+- 可以纳入后续治理项
