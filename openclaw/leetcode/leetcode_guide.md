*作者：大胖超 😜*

---

## 🎯 一、刷题前的准备

### 1. 心态建设
- **别想一口吃成胖子**：每天 1-2 题，坚持比突击重要
- **接受卡壳**：一道题想 1-2 小时很正常，想不出来就看题解
- **不要比较**：别人刷 500 题跟你没关系，按自己的节奏来

### 2. 技术准备
- 选一门主力语言（Python/Java/C++ 推荐）
- 准备好 IDE 或直接用 LeetCode 在线编辑器
- 准备一个笔记系统（Notion/语雀/本地 Markdown 都行）

---

## 📋 二、刷题顺序（按这个来！）

### 阶段 1：基础入门（1-50 题）
| 类型 | 推荐题目数 | 目标 |
|------|-----------|------|
| 数组/字符串 | 15 题 | 熟悉基本操作 |
| 链表 | 8 题 | 指针操作 |
| 栈/队列 | 8 题 | 理解数据结构 |
| 哈希表 | 10 题 | 空间换时间 |
| 简单二分 | 5 题 | 边界条件 |
| 简单递归 | 4 题 | 递归思维 |

### 阶段 2：核心算法（51-200 题）
- **排序算法**：快速排序、归并排序、堆排序
- **二叉树**：遍历、BST、递归/迭代
- **回溯算法**：排列、组合、子集
- **动态规划**：从简单 DP 开始（爬楼梯、打家劫舍）
- **贪心算法**：区间问题、分配问题
- **图论基础**：BFS、DFS、并查集

### 阶段 3：进阶提升（200+ 题）
- 复杂动态规划
- 线段树、树状数组
- 高级图论（最短路径、最小生成树）
- 字符串算法（KMP、Trie）
- 数学/位运算技巧

## 🧠 三、解题方法论

### 五步解题法
```
1️⃣ 读题：搞清楚输入输出、边界条件、约束
2️⃣ 举例：手动算几个例子，找规律
3️⃣ 思路：想暴力解 → 优化 → 最优解
4️⃣ 编码：先写对，再写好
5️⃣ 复盘：看题解，学更好的写法
```

### 常见套路速查
| 题型 | 第一反应 |
|------|---------|
| 求最值 | 动态规划/贪心 |
| 所有组合 | 回溯 |
| 有序数组查找 | 二分 |
| 子数组/子串 | 滑动窗口 |
| Top K 问题 | 堆 |
| 连通性问题 | 并查集 |
| 最短路径 | BFS |
| 括号匹配 | 栈 |

## 🔄 五、复习策略（超重要！）

### 艾宾浩斯复习法
- 第 1 次复习：做完当天晚上
- 第 2 次复习：第 2 天
- 第 3 次复习：第 4 天
- 第 4 次复习：第 7 天
- 第 5 次复习：第 15 天

### 标记系统
- ✅ 一次 AC，完全理解
- ⚠️ 看题解才懂，需要复习
- ❌ 完全没思路，重点标记
