# code-review

## 定位

通用代码审查 Skill，负责在进入专项审查前先做正确性、安全性、可维护性和测试覆盖的基础筛查。

## 适用场景

- 审查 PR、分支差异、单文件改动
- 做首次质量把关
- 作为其他专项 Skill 的入口

## 核心关注点

- 正确性与边界条件
- 错误处理与资源释放
- 基础安全问题
- 性能和可维护性风险
- 测试缺口与回归风险

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `RULES.md`：通用审查规则摘要
- `CHECKLISTS/`：语言相关核对项
- `EXAMPLES/`：示例占位目录
- `TEMPLATES/`：报告模板

## 协作边界

- 发现 C++ 生命周期、并发、协议、性能热点后，应转交对应专项 Skill 深挖。
