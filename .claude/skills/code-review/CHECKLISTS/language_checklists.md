# 通用语言审查清单

## 通用项

- [ ] 空输入、边界值、异常路径是否覆盖
- [ ] 资源释放与错误处理是否完整
- [ ] 是否引入明显安全问题
- [ ] 是否存在明显性能退化风险
- [ ] 测试是否覆盖新增行为

## C++

- [ ] 所有权、RAII、析构路径是否清晰
- [ ] 裸指针、悬垂引用、拷贝成本是否可接受
- [ ] public API 和 ABI 影响是否说明

## Python

- [ ] 资源是否通过上下文管理
- [ ] 异常捕获是否具体
- [ ] 可变默认参数和动态执行是否避免

## TypeScript / JavaScript

- [ ] 类型、空值和异步路径是否清晰
- [ ] 是否避免 `any` 和不安全 DOM 注入

## Go

- [ ] `err` 是否检查
- [ ] goroutine 生命周期是否可控
- [ ] `defer` 和共享数据并发访问是否安全
