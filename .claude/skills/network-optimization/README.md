# network-optimization

## 定位

网络优化 Skill，聚焦连接管理、超时与重试、带宽、传输效率、尾延迟和连接稳定性。

## 适用场景

- 网关、RPC、行情分发、客户端接入优化
- 网络尾延迟升高或连接不稳定
- 编解码、批处理、重试和连接池行为审查

## 核心关注点

- 往返次数与协议选择
- 连接复用、重试、超时和背压
- 编解码和传输放大成本
- 丢包、乱序、重放和稳定性

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：网络和瓶颈清单
- `EXAMPLES/`：网络问题示例
- `TEMPLATES/`：网络优化模板占位目录

## 协作边界

- 协议语义和版本兼容问题可联动 `protocol-validate`
- 通用性能定位可联动 `performance-bottleneck-identification`
