# compliance-review

## 定位

合规审查 Skill，聚焦审计追踪、控制项、报表一致性、权限记录、保留策略和监管要求映射。

## 适用场景

- 金融、支付、审计链路改动
- 合规报表、留痕、对账、控制检查
- 上线前做监管与内控视角复核

## 核心关注点

- 控制项是否落地
- 审计记录是否完整可追溯
- 敏感操作是否可解释
- 数据保留、报送、追踪是否满足要求

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：合规核对项
- `EXAMPLES/`：示例占位目录
- `TEMPLATES/`：报告模板

## 协作边界

- 漏洞和攻击面问题优先交 `security-review`
- 数值、风险和数据口径问题可联动 `risk-check`
