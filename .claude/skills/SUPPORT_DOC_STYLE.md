# Claude Skills 支持文档风格约定

## README.md

统一使用：

1. 定位
2. 适用场景
3. 核心关注点
4. 目录说明
5. 协作边界

## SKILL.md

统一使用：

1. 目标
2. 适用条件
3. 审查重点
4. 输入期望
5. 输出要求
6. 协作边界

## CHECKLISTS/

统一使用：

1. 使用时机（可选）
2. 关键检查项
3. 阻塞问题示例（可选）

重点是短、小、可执行。

## EXAMPLES/

统一使用：

1. 示例输入
2. 期望输出

尽量少写 CLI 安装、重启、crontab、hook 说明。

## TEMPLATES/

统一使用：

1. 最小可复用结构
2. 常用字段占位符
3. 不写冗长解释
