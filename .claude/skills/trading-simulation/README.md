# trading-simulation

## 定位

交易模拟审查 Skill，聚焦撮合规则、执行延迟、滑点、手续费、订单生命周期和模拟真实性。

## 适用场景

- 交易模拟器、撮合器、订单执行模型审查
- 回测与模拟结果偏差较大
- 需要验证执行层假设是否可信

## 核心关注点

- 撮合规则与优先级
- 延迟、滑点、手续费和部分成交
- 订单状态迁移与取消修改
- 模拟结果与真实市场差距

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：规则摘要
- `EXAMPLES/`：使用示例
- `TEMPLATES/`：模板占位目录

## 协作边界

- 纯回测统计问题可联动 `quant-backtest`
- 协议和市场数据输入问题可联动 `protocol-validate` 或 `market-data-validate`
