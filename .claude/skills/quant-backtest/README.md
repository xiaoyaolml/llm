# quant-backtest

## 定位

回测验证 Skill，聚焦策略逻辑、样本选择、时间对齐、未来函数、统计偏差和结果可解释性。

## 适用场景

- 量化策略回测审查
- 回测结果异常、收益过高或稳定性不足
- 需要验证数据与模型假设是否合理

## 核心关注点

- 时间对齐与未来函数
- 样本切分与过拟合
- 滑点、手续费、成交假设
- 统计可信度与结果解释

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：回测清单占位目录
- `EXAMPLES/`：回测示例占位目录
- `TEMPLATES/`：回测模板占位目录

## 协作边界

- 市场数据质量问题可联动 `market-data-validate`
- 交易模拟真实性问题可联动 `trading-simulation`
