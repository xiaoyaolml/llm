# defi-protocol-review

## 定位

DeFi 协议审查 Skill，聚焦资产流转、状态机、权限边界、预言机、流动性和升级风险。

## 适用场景

- 智能合约协议、资金池、清算、治理逻辑审查
- 代币经济、预言机或跨链交互评估
- 协议上线前或审计复核

## 核心关注点

- 资产流和状态变更是否安全
- 权限、治理和升级边界
- 预言机、精度、价格操纵风险
- 事件、审计和监控能力

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：规则摘要
- `EXAMPLES/`：示例
- `TEMPLATES/`：模板占位目录

## 协作边界

- 通用安全问题可联动 `security-review`
- API 或协议暴露面可联动 `api-design-review`
