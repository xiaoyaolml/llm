# cloud-native-review

## 定位

云原生审查 Skill，聚焦容器化、配置管理、部署方式、弹性、故障恢复和平台交付质量。

## 适用场景

- 服务容器化或部署方案评审
- 配置、资源、健康检查、弹性策略调整
- 平台迁移或交付前检查

## 核心关注点

- 镜像、配置、依赖和权限边界
- 健康检查、超时、重试、退化策略
- 扩缩容、资源限制、容量规划
- 部署、回滚、观测和演练准备

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：云原生规则摘要
- `EXAMPLES/`：使用示例
- `TEMPLATES/`：模板占位目录

## 协作边界

- 监控和告警细节可联动 `system-monitoring`
- 安全边界可联动 `security-review`
