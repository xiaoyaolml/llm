# risk-check

## 定位

风险计算审查 Skill，聚焦数值稳定性、边界条件、模型假设、口径一致性和风险结果可解释性。

## 适用场景

- 风险模型、限额、估值或压力测试改动
- VaR、Greeks、蒙特卡洛、相关性矩阵等计算链路审查
- 风险结果异常或回归偏差排查

## 核心关注点

- 精度、舍入、溢出、收敛
- 输入数据和假设是否合理
- 风险指标口径与报表一致性
- 结果验证和回归测试

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `RULES.md`：风险规则摘要
- `CHECKLISTS/`：风险计算核对清单
- `EXAMPLES/`：示例占位目录
- `TEMPLATES/`：报告模板

## 协作边界

- 数据源质量问题可联动 `market-data-validate`
- 回测或策略验证问题可联动 `quant-backtest`
