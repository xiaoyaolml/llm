# ml-model-validate

## 定位

模型验证 Skill，聚焦训练/推理一致性、特征工程、数据泄漏、漂移、偏差和线上监控。

## 适用场景

- 训练和推理链路审查
- 模型效果异常、漂移或稳定性下降
- 上线前做模型验证复核

## 核心关注点

- 数据与标签质量
- 特征工程一致性和泄漏
- 指标选择、漂移和偏差
- 线上监控与回归验证

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：模型规则摘要
- `EXAMPLES/`：示例
- `TEMPLATES/`：模板占位目录

## 协作边界

- 风险口径问题可联动 `risk-check`
- 线上可观测性问题可联动 `system-monitoring`
