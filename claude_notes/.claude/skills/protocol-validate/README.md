# protocol-validate

## 定位

协议验证 Skill，聚焦消息结构、编解码、会话状态、错误语义、版本兼容和边界条件处理。

## 适用场景

- FIX、自定义二进制、RPC 或流式协议实现审查
- 编码器、解码器、会话管理、重传处理审查
- 协议升级和兼容性评估

## 核心关注点

- 报文边界、字段语义和版本兼容
- 半包、坏包、乱序、重放和超时处理
- 会话状态、序列号和重传
- 编解码性能与错误语义

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：协议核对项占位目录
- `EXAMPLES/`：协议问题示例占位目录
- `TEMPLATES/`：协议审查模板占位目录

## 协作边界

- 网络链路效率问题可联动 `network-optimization`
- API 语义问题可联动 `api-design-review`
