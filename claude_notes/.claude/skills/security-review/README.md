# security-review

## 定位

安全审查 Skill，聚焦输入验证、认证授权、敏感信息、审计、依赖风险和配置暴露面。

## 适用场景

- 身份认证、授权、会话、密钥管理改动
- 对外接口、输入处理、配置安全审查
- 上线前安全评估或事故复盘

## 核心关注点

- 注入、越权、敏感信息泄露
- 密钥、令牌、凭据和加密使用
- 审计日志、异常行为和追踪
- 依赖风险与默认配置安全

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：安全检查清单
- `EXAMPLES/`：安全审查示例
- `TEMPLATES/`：模板占位目录

## 协作边界

- 合规控制与审计要求可联动 `compliance-review`
- 协议与 API 暴露面可联动 `protocol-validate` 或 `api-design-review`
