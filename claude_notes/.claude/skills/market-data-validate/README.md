# market-data-validate

## 定位

市场数据验证 Skill，聚焦完整性、准确性、一致性、时效性、顺序性和异常数据处理。

## 适用场景

- 行情接入、归档、清洗、分发链路审查
- 排查坏包、重复包、乱序、延迟数据问题
- 做数据质量和口径一致性核对

## 核心关注点

- 缺失、重复、乱序、延迟与脏数据
- 时间戳、价格、数量单位和口径
- 快照与增量一致性
- 数据源切换与回放验证

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：市场数据清单占位目录
- `EXAMPLES/`：数据问题示例占位目录
- `TEMPLATES/`：质量报告模板占位目录

## 协作边界

- 协议层格式问题可联动 `protocol-validate`
- 策略和回测使用问题可联动 `quant-backtest`
