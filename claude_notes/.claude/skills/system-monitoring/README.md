# system-monitoring

## 定位

系统监控 Skill，聚焦指标、日志、trace、告警、SLO 和 runbook，确保系统可观测、可诊断、可交付。

## 适用场景

- 新服务上线前审查
- 排障过程中补观测点
- 监控、告警、仪表板改动审查

## 核心关注点

- 核心指标是否齐全
- 日志和 trace 是否能定位问题
- 告警是否贴近 SLO 与用户影响
- runbook、容量和演练是否完善

## 目录说明

- `SKILL.md`：技能定义与输出要求
- `CHECKLISTS/`：监控规则摘要
- `EXAMPLES/`：使用示例
- `TEMPLATES/`：模板占位目录
